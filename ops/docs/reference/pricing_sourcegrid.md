## SourceGrid Labs (English / USD)

### Package overview

| Package           | Includes                                                                 | Price (USD) |
| ----------------- | ------------------------------------------------------------------------- | ----------- |
| Small Website     | 6–8 pages, mobile-friendly, contact form, basic admin.                    | 4,000       |
| Business System   | Website plus online system (login, dashboard, one core module).           | 15,000      |
| Mobile App Suite  | iOS + Android app, backend system, and website.                            | 30,000      |

All packages are fixed-price; costs do not increase if technical work expands.

### Package details

**🌐 Small Website — $4,000**

- 6–8 page site (home, about, services, pricing, contact, optional blog/news)
- Mobile responsive design
- Contact form routed to client email
- Basic SEO setup
- Admin login for copy/image edits
- Hosting setup + deployment

Typical clients: cafés, consultants, local services.

**⚙️ Business System — $15,000**

- Everything in Small Website
- Customer login area
- Staff dashboard
- One core module (booking, rental, or workflow tool)
- Online payments + automatic email confirmations
- CSV/Excel export

Typical clients: boutique hotels, rental shops, clinics.

**📱 Mobile App Suite — $30,000**

- Everything in Business System
- Branded iOS + Android app (single codebase)
- Customer login + profiles
- Push notifications
- In-app booking/ordering
- App Store + Play Store submission

Typical clients: delivery/logistics, loyalty programs, mobile-first orgs.
